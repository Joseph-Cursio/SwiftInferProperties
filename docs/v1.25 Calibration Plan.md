## 2. The workstream

### Workstream A — Index direction-op idempotence veto

**What:** Extend `IdempotenceTemplate.directionLabelCounterSignal(for:)` (V1.10.1) from `-15` (current single-side fire) to `-25` (full veto-equivalent) when **both** of:

1. The function's name starts with `index`, `bucket`, or `word` (or is exactly one of those names — handles overload variants).
2. The first-parameter label is in `DirectionLabels.curated` (already required by V1.10.1).

The combined match identifies the cycle-21 reject class: index-advance functions where the direction label fires V1.10.1 at -15 but the name pattern adds high confidence that the function isn't actually idempotent (it's a positional cursor advance, not a fixed-point operation).

**Cycle-21 origin:** the cycle-20 sample (V1.20.C) measured 0/4 = 0% on idempotence non-lifted (cycle-17 was 0/4 = 0% too — 5-cycle-flat). The cycle-21 findings doc identifies the dominant residual reject class.

**Implementation site:** modify `IdempotenceTemplate.directionLabelCounterSignal(for:)` in `Sources/SwiftInferTemplates/IdempotenceTemplate.swift` (or whichever file currently houses this method — V1.10.1 hoist location). Add a name-prefix gate to the magnitude bump.

**Score posture:** when both name-prefix + direction-label match, fire at -25 → score 30 + 5 - 25 = 10 → Suppressed. Closes the class. When only direction-label matches (no index/bucket/word prefix), preserve V1.10.1's -15 magnitude — rate-stability on non-index direction-ops.

**Mechanism class:** extension of class 6 (parameter-label direction-counter, V1.10.1 lineage) — magnitude variation with name-prefix gate. Same shape as V1.22.B's both-sides direction full-veto extension on round-trip.

**Cost:** ~1-2 hours. Single-method change in IdempotenceTemplate.swift; new curated set `IndexAdvanceNamePrefixes.curated = {index, bucket, word}` in core (or inline if small enough); ~8 unit tests covering the matrix (name-prefix-match × label-match, single side only, etc.).

**Plan-vs-actual projection:** **closes ~13-15 OC candidates** (the dominant idempotence non-lifted reject class identified in cycle-21 findings). Total v1.25 closure projection: -13 to -15 candidates. Net surface: 130 → ~115-117 (-10 to -12%).
